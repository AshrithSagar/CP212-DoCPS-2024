# String functions

Implement the following functions in assembly:

```c
int mystrlen(char *s);

char *mystrcpy(char*dest, char *src);

int mystrcmp(char *s1, char*s2);
```

## Deliverables

- `string.h`: function declarations (prototypes)

- `string.s`: function definitions in assembly

- `main.c`: test code
